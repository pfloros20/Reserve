Το “Reserve” είναι μια εφαρμογή η οποία απευθύνεται σε καταστηματάρχες και πελάτες καταστημάτων, όπως καφετέριες και εστιατόρια. Οι καταστηματάρχες θα μπορούν να έχουν μια σελίδα για το κατάστημά τους με πληροφορίες για αυτό, όπως τοποθεσία, ώρες λειτουργίας, προσφορές του μαγαζιού, πιθανά events που διοργανώνονται, φωτογραφίες του μαγαζιού, και το μενού. Μέσω της ιστοσελίδας οι ιδιοκτήτες έχουν την δυνατότητα να διαχειρίζονται κρατήσεις, να επεξεργάζονται την κατάσταση διαθεσιμότητας των τραπεζιών ανά πάσα χρονική στιγμή και να διοργανώνουν εκδηλώσεις που θα δημοσιεύονται στην εφαρμογή στην οποία θα είναι ορατές στους χρηστές. Οι πελάτες θα μπορούν να αναζητήσουν καταστήματα είτε ανάλογα με το πόσο κοντά είναι σε μια τοποθεσία, είτε μέσω ονόματος ή χρησιμοποιώντας κάποια φίλτρα. Επιπλέον, μπορούν να επιλέξουν τα αγαπημένα τους καταστήματα ώστε να τα βρίσκουν γρήγορα. Επιπροσθέτως, θα μπορούν να πληροφορηθούν για τα καταστήματα μέσω της εφαρμογής, να δουν την κίνηση σε κάθε κατάστημα, να κάνουν κράτηση και να γράφουν κριτικές για τα μαγαζιά που επισκέφτηκαν.
